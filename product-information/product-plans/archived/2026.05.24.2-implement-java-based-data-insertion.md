# Plan: Local Seed Data via Spring ApplicationRunner

**Date:** 2026-05-24

## Overview

Create a single Java class, `DataSeeder`, that seeds a stable test user (with task
lists and tasks) into the local database on first boot. It is guarded by
`@Profile("local")` so it never runs in `qa` or `prod`.

---

## Why Not Liquibase Changesets

Liquibase changesets are SQL — they have no access to Java logic. Two things in this
app require Java at insert-time:

1. **BCrypt** — `password_hash` must be produced by `PasswordEncoder.encode()`. You
   could pre-compute a BCrypt hash offline and hardcode it, but that means a known
   password is committed to version control and the hash becomes stale if the work
   factor ever changes.
2. **Field-level encryption** — `task_lists.name`, `tasks.title`, and `tasks.notes`
   are encrypted by `Encryptors.text(ENCRYPTION_PASSWORD, ENCRYPTION_SALT)` at the
   application layer. `ENCRYPTION_PASSWORD` and `ENCRYPTION_SALT` come from `.env`
   and vary per developer. A hardcoded ciphertext in a Liquibase file would be
   corrupted on every other machine.

The only correct fix for both issues is running the real application services.

---

## Approach: `@Profile("local") ApplicationRunner`

Spring Boot's `ApplicationRunner` interface provides a `run()` method that is called
once, after the full application context is ready. By annotating the class with
`@Profile("local")`, Spring will not even instantiate it in `qa` or `prod` — it is
completely invisible to those environments.

### Idempotency

The seeder looks up the seed email before doing anything. On every subsequent
restart it finds the user and exits immediately. No duplicate data is ever created.

### No email side-effect

`UserService.register()` sends a verification email and sets `emailVerified = false`.
The seeder bypasses this by calling `userRepository.save()` directly — the same
path used internally by `UserService` — and sets `emailVerified = true` inline.
`UserRepository.setEmailVerified()` also exists if you prefer calling `register()`
first and patching afterward, but the direct save is simpler.

## File to Create

### `backend/src/main/java/com/killeen/taskflow/config/DataSeeder.java`

```java
package com.killeen.taskflow.config;

import java.time.OffsetDateTime;
import java.time.ZoneOffset;
import java.util.List;

import org.springframework.boot.ApplicationArguments;
import org.springframework.boot.ApplicationRunner;
import org.springframework.context.annotation.Profile;
import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.stereotype.Component;

import com.killeen.taskflow.components.task.model.CreateTaskRequest;
import com.killeen.taskflow.components.task.service.TaskService;
import com.killeen.taskflow.components.tasklist.model.CreateTaskListRequest;
import com.killeen.taskflow.components.tasklist.model.TaskList;
import com.killeen.taskflow.components.tasklist.service.TaskListService;
import com.killeen.taskflow.components.user.model.User;
import com.killeen.taskflow.components.user.repository.UserRepository;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;

@Profile("local")
@Component
@Slf4j
@RequiredArgsConstructor
public class DataSeeder implements ApplicationRunner {

    // Credentials are deliberately simple — this account is for local use only.
    private static final String SEED_EMAIL    = "seed@local.dev";
    private static final String SEED_PASSWORD = "SeedPass123!";

    private final UserRepository  userRepository;
    private final PasswordEncoder passwordEncoder;
    private final TaskListService taskListService;
    private final TaskService     taskService;

    // -------------------------------------------------------------------------
    // Entry point
    // -------------------------------------------------------------------------

    @Override
    public void run(ApplicationArguments args) {
        if (userRepository.findByEmail(SEED_EMAIL).isPresent()) {
            log.info("DataSeeder: seed user already exists — skipping.");
            return;
        }

        User user         = createSeedUser();
        List<TaskList> lists = createSeedLists(user.getId());
        createSeedTasks(user.getId(), lists);

        log.info("============================================================");
        log.info("DataSeeder: seed data inserted.");
        log.info("  email:    {}", SEED_EMAIL);
        log.info("  password: {}", SEED_PASSWORD);
        log.info("============================================================");
    }

    // -------------------------------------------------------------------------
    // User
    // -------------------------------------------------------------------------

    private User createSeedUser() {
        OffsetDateTime now = OffsetDateTime.now(ZoneOffset.UTC);
        User user = User.builder()
                .email(SEED_EMAIL)
                .passwordHash(passwordEncoder.encode(SEED_PASSWORD))
                .displayName("Seed User")
                .emailVerified(true)   // bypass the email-verification flow
                .createdAt(now)
                .updatedAt(now)
                .build();
        return userRepository.save(user);
    }

    // -------------------------------------------------------------------------
    // Task lists
    // -------------------------------------------------------------------------

    private List<TaskList> createSeedLists(Long userId) {
        return List.of(
                createList(userId, "Work",     "#6366f1"),
                createList(userId, "Personal", "#10b981"),
                createList(userId, "Shopping", "#f59e0b")
        );
    }

    private TaskList createList(Long userId, String name, String color) {
        CreateTaskListRequest req = new CreateTaskListRequest();
        req.setName(name);
        req.setColor(color);
        return taskListService.createTaskList(userId, req);
    }

    // -------------------------------------------------------------------------
    // Tasks
    // -------------------------------------------------------------------------

    private void createSeedTasks(Long userId, List<TaskList> lists) {
        Long workId     = lists.get(0).getId();
        Long personalId = lists.get(1).getId();
        // lists.get(2) = Shopping — left empty intentionally to simulate an empty list

        // Work list
        createTask(userId, workId, "Review Q2 roadmap presentation", "Check slides before the all-hands.");
        createTask(userId, workId, "Fix login page bug",              null);
        createTask(userId, workId, "Write release notes for v1.3",    null);
        createTask(userId, workId, "Schedule 1-on-1 with Alex",       null);

        // Personal list
        createTask(userId, personalId, "Morning jog 5 km",              null);
        createTask(userId, personalId, "Book dentist appointment",       "Ask about the crown replacement.");
        createTask(userId, personalId, "Buy birthday gift for Sarah",    null);
        createTask(userId, personalId, "Renew car registration",         null);

        // No list (floaters)
        createTask(userId, null, "Read a chapter of the book", null);
        createTask(userId, null, "Call mum",                   null);
    }

    private void createTask(Long userId, Long listId, String title, String notes) {
        CreateTaskRequest req = new CreateTaskRequest();
        req.setTitle(title);
        req.setNotes(notes);
        req.setListId(listId);
        taskService.createTask(userId, req);
    }
}
```

## How It All Fits Together at Runtime

```
App starts (spring.profiles.active=local)
  └─ Spring context loads DataSeeder (excluded in qa/prod)
       └─ ApplicationRunner.run() fires after context is ready
            ├─ findByEmail("seed@local.dev") → empty → proceed
            ├─ userRepository.save(...)
            │    └─ BCrypt hash computed by PasswordEncoder bean
            │    └─ emailVerified = true (no email sent)
            ├─ taskListService.createTaskList(...)  ×3
            │    └─ TaskListEncryptionHelper.encrypt() called automatically
            └─ taskService.createTask(...)  ×10
                 └─ TaskEncryptionHelper.encryptTask() called automatically
                 └─ position computed as COUNT(*) per list (existing service logic)

Log output:
  ============================================================
  DataSeeder: seed data inserted.
    email:    seed@local.dev
    password: SeedPass123!
  ============================================================
```

On every subsequent restart, `findByEmail` returns the existing user and `run()`
returns immediately.
