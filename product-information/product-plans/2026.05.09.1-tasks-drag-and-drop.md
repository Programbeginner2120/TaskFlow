# Implementation Plan

1. Get the frontend drag and drop working on task lists

2. Add liquibase script to give tasks nullable positions

```sql
ALTER TABLE public.tasks
ADD COLUMN BIGINT;
```

3. Run the liquibase change
4. Run mybatis generator to make the change effective

```bash
.\gradlew mybatisGenerator
```

5. Update create task such that each time a new task is added to a list, it adds with the incremented position
6. Update update task such that each time a task is moved in a list, it updates the positions for that task cascading
7. Update delete task such that each time a task is deleted in a list, it updates the positions for that task cascading
